## 📥 Installation

To run locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/shadow-slave/workPortfolio.git
   cd aswin-portfolio
   ```

2. Open `index.html` in your browser:

   ```bash
   open index.html  # macOS
   start index.html # Windows
   ```
